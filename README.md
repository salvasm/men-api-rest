# Node.js API Rest

## Built With
- Node.js
- Express
- MongoDB

## Getting Started
1. Install MongoDB
2. Clone the repo:
```
git clone https://github.com/salvasm/node-api-rest.git
```

3. Install NPM packages:
```
npm install
```

4. Build and run server command:
```
npm run build && npm run start
```
