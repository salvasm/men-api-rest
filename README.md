# Node.js API Rest
API Rest boilerplate

## Built With
- Node.js
- Express
- MongoDB


## Integrated
- Basic auth routes
- CRUD class
- jsonwebtoken
- sessions
- ACL
- Error Handling

## Getting Started
1. Install MongoDB
2. Clone the repo:
```
git clone https://github.com/salvasm/node-api-rest.git
```

3. Install NPM packages:
```
npm install
```

4. Build and run server command:
```
npm run build && npm run start
```

## Routes
- /api
    - /auth
        - POST /authentication
        - POST /logout
    - /user
        - GET /
        - POST /
        - GET /:id
        - PUT /:id
        - DELETE /:id
